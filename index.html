How to play

Move cleanly. Collect fast.

  • Use WASD or Arrow Keys to move.
  • Press Space or J to attack.
  • Press 1 through 5 for character skills.
  • Start Game opens the gate lobby where story stages and boss-only gates can be chosen.
  • Defeat enough enemies to summon each stage boss.
  • Use Loadout between runs to buy, equip, and upgrade weapons or characters.
  • Your progress autosaves in this browser. Save File and Load File move it between machines.
